# webServerTools

#### 目录详细信息
```text
├─apps                实际处理业务代码
├─common              常用的项目公共代码
│  ├─error            项目同意错误信息
│  ├─global           项目全局参数
│  ├─logger           项目日志
│  └─response         项目统一响应
├─config              项目配置
├─controllers         控制层  
├─deploy              项目部署文件
├─middlewares         服务中间件
├─models              项目模型文件
├─routes              路由
├─services            常用组件服务目录
│  ├─cache            缓存组件
│  ├─db               数据连接
│  └─kafka            kafka连接
├─static              静态文件存放目录
└─utils               常用工具函数


request -> router -> controller -> apps -> response
```
